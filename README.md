# dezlymacauley-dot-com
_______________________________________________________________________________

## Required Tools

- [mise](https://mise.jdx.dev/)
_______________________________________________________________________________

## Setup Guide

Clone the repository

```
git clone https://github.com/dezlymacauley/dezlymacauley-dot-com.git
```
_______________________________________________________________________________

Give mise permission to read from the `mise.toml` file

```
mise trust
```
_______________________________________________________________________________

Ensure that the tools listed in the `mise.toml` file are installed.

```
mise install
```
_______________________________________________________________________________

Run the `setup` script from the `package.json`

```sh
bun run setup
```
